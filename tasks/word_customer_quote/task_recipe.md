# Task Recipe: Word Customer Quote

Prepare or revise a customer quote or proposal in Word with clear control over wording, placement, and save behavior.

## Best Used When

- A salesperson must update a quote
- A proposal section needs wording changes
- A document should be converted to PDF after review

## Inputs

- Document path or name
- Section or heading to edit
- Client name and updated offer details
- Save or export instruction

## Success Conditions

- The right section was updated
- Layout remains readable
- Client name, dates, and totals are correct
- Save or export behavior matches the instruction

## Prompt Examples

- `Wordで既存の見積書を開き、顧客名と金額を更新して、PDF出力前で止めて。`
- `提案書の「導入効果」セクションだけ文章を整えて、元ファイルは上書きせず別名保存して。`

## Tips

- Mention heading names instead of “somewhere in the middle”
- Say whether formatting should stay untouched or can be adjusted
- If a PDF is needed, specify whether both Word and PDF should be kept
