# Task Recipe: Teams Follow-up Summary

Use Teams content to prepare a concise follow-up summary with decisions, next actions, and owners.

## Best Used When

- A meeting chat contains the latest decisions
- A user needs a follow-up note after a call
- Action items must be pulled from conversation history

## Inputs

- Team or chat name
- Meeting or date reference
- Output destination
- Draft-only or send instruction

## Success Conditions

- The correct chat or meeting thread was opened
- The follow-up summary reflects visible discussion points
- Actions, owners, and due dates are captured where possible

## Prompt Examples

- `Teamsで今日の打ち合わせチャットを見て、決定事項と宿題をWordにまとめて。`
- `会議後フォローの要点をTeamsから拾って、Outlookの下書きメールに入れて。送信はしないで。`

## Tips

- Mention the output location: Teams post, Word, Outlook draft, or plain clipboard text
- Ask for "visible items only" if the source thread is long
- If dates matter, ask for owner and deadline formatting explicitly
