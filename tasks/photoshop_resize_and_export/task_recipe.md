# Task Recipe: Photoshop Resize And Export

Adjust image size or crop in Photoshop and export a clean deliverable without damaging the source file.

## Best Used When

- Marketing images need channel-specific sizes
- A photo needs crop and export
- Web assets must be produced from a source file

## Inputs

- Source image path
- Target dimensions
- Crop rule
- Output format and save behavior

## Success Conditions

- The correct image is open
- Size or crop matches the request
- Exported file exists in the requested format
- Original source remains intact unless overwrite is explicit

## Prompt Examples

- `Photoshopでこの画像を1200x628に調整して、PNGで書き出して。元画像は残して。`
- `商品画像を正方形にトリミングして、Web用JPEGを書き出して。`

## Tips

- Include exact pixel size when possible
- Say whether the crop should preserve faces, product center, or full margins
- Prefer “export a copy” when the original must survive
