# Task Recipe: PDF Merge and Bookmark

Combine several PDFs and add a bookmark-friendly structure for sharing or review.

## Best Used When

- Proposal or bid documents come from several source PDFs
- A technical packet must be assembled into one file
- A merged PDF needs a logical reading order

## Inputs

- Source folder or file list
- Desired merge order
- Output file name
- Bookmark requirement

## Success Conditions

- The correct files were opened and merged in the requested order
- The output PDF was created with a clear name
- Bookmark or section markers were added when requested

## Prompt Examples

- `Acrobatで3つのPDFを結合して、表紙、本文、見積の順に並べて保存して。`
- `提案資料PDFをまとめて、主要セクションにしおりを付けて。`

## Tips

- Give the exact file order instead of saying "merge these"
- If the final PDF is for customers, mention the preferred output name
- If bookmarks are needed, describe the section titles to use
