# Microsoft Edge App Profile

Use Edge like a standard Windows browser with focus on tabs, downloads, forms, and web navigation.

## Purpose

Edge is useful for research, internal portals, downloads, CRM systems, and web-based back-office tasks.

## Main Screens

- Tab strip
- Address bar
- Web page canvas
- Download shelf or hub
- Print or save dialogs

## Typical Tasks

- Search and compare web sources
- Log in to web tools
- Fill forms
- Download files and verify completion

## Important Shortcuts

- `Ctrl+L`
- `Ctrl+T`
- `Ctrl+W`
- `Ctrl+F`
- `Ctrl+J`

## Common Pitfalls

- Focusing the page instead of the address bar
- Pop-ups or cookie banners blocking controls
- Mistaking one tab for another
- Submitting a form too early

## Operational Notes

- Treat web apps as normal GUI targets and use optional browser adapters only when a connected page is available
- Prefer address bar, page titles, visible headings, and form labels as stable anchors
- Complete form submission and external posting when they are clearly requested
- After downloads, verify the file name and destination before continuing

## Prompt Tips

- Mention site name, target page, and final success signal
- State whether the task ends at draft, populated form, or confirmed submission
- For research tasks, tell the agent what facts to extract instead of only saying “look it up”
