# File Explorer App Profile

Organize files and folders carefully while minimizing accidental move, rename, delete, or overwrite actions.

## Purpose

File Explorer is best for desktop cleanup, download sorting, folder creation, file renaming, and archiving.

## Main Screens

- Navigation pane
- File list
- Address bar
- Search box
- Context menus and file operation dialogs

## Typical Tasks

- Open a folder
- Sort and group files
- Rename files or folders
- Move or copy files
- Compress to ZIP

## Important Shortcuts

- `Ctrl+L`
- `Ctrl+C`
- `Ctrl+X`
- `Ctrl+V`
- `F2`
- `Alt+Up`

## Common Pitfalls

- Moving files to the wrong folder
- Renaming the wrong selection
- Overwriting existing files during copy or move
- Losing context when multiple Explorer windows are open

## Operational Notes

- Confirm folder path before any move, rename, or delete operation
- Prefer creating clearly named destination folders before bulk moves
- Use list columns, sort order, and selection count as validation cues
- Delete, overwrite, move, or rename files when explicitly requested and the target folder is clear

## Prompt Tips

- Include source folder, destination folder, naming pattern, and whether duplicates should be skipped, overwritten, or reviewed
- For cleanup tasks, define the grouping rule such as by extension, project, month, or client
- If the task should stop before deletion, say so explicitly
