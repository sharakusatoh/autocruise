# Excel App Profile

## Purpose

Spreadsheet editing and repetitive business workflows with clear workbook and sheet context.

## Main Screens

- Workbook grid
- Ribbon
- Formula bar
- Save and file dialogs

## Typical Tasks

- Edit cells
- Sort and filter data
- Export or save workbooks

## UI Structure

- Ribbon tabs
- Name box and formula bar
- Sheet tabs
- Status bar

## Important Menus

- File
- Home
- Data
- Formulas

## Important Shortcuts

- `Ctrl+S`
- `Ctrl+Shift+L`
- `Ctrl+F`
- `Alt`

## Common Pitfalls

- Wrong workbook or wrong sheet focus
- Filter scope ambiguity
- Unexpected save prompts and similar workbook names

## Version Differences

- Ribbon grouping and localized labels vary

## Operational Notes

- Validate workbook title and sheet context before edits
- Save or replace files when the requested output clearly calls for it

## Common Success Patterns

- Use name box and ribbon cues to confirm selection
- Favor shortcuts for stable commands
