# Task Recipe: Outlook Follow-up Email

Draft a follow-up email from Outlook while keeping the requested send behavior explicit.

## Best Used When

- A salesperson needs a polite follow-up mail
- A reply draft should summarize an earlier meeting
- A reminder mail must be prepared with an attachment

## Inputs

- Recipient name
- Subject line
- Main points to include
- Attachment requirement
- Draft-only or send-now instruction

## Success Conditions

- The draft addresses the right person
- Subject and body match the request
- Attachments are correct if required
- The run stops at draft or send exactly as requested

## Prompt Examples

- `OutlookでA社の田中さん宛てのフォローアップメールを下書きして。送信はしないで。`
- `先週の打ち合わせのお礼メールをOutlookで作成し、資料を添付して送信して。`

## Tips

- If sending is not required, explicitly say “draft only”
- Mention attachment file names and expected location
- Include desired tone such as formal, concise, or friendly-business
