# Codex Desktop App Profile

## Purpose

Desktop coding assistant with chat, workspace context, and code review flows.

## Main Screens

- Conversation view
- Composer input
- Workspace or file context

## Typical Tasks

- Enter instructions
- Review generated content
- Navigate history

## UI Structure

- Conversation list
- Main transcript
- Composer

## Important Menus

- File
- View
- Help

## Important Shortcuts

- `Ctrl+L`
- `Ctrl+K`
- `Ctrl+Enter`

## Common Pitfalls

- Wrong thread focus
- Sending before verifying prompt contents

## Version Differences

- Pane layout may evolve

## Operational Notes

- Send messages when the destination and requested action are clear
- Confirm active thread before sending

## Common Success Patterns

- Verify transcript context and composer focus
- Prefer keyboard shortcuts for sending only after confirmation
