# Task Recipe: Photoshop Banner Text Update

Open a Photoshop asset, update visible text, and export a clean delivery image without damaging the source PSD.

## Best Used When

- A sales or marketing banner needs a headline change
- A promotional image must be updated for a specific campaign
- A localized image variant is required quickly

## Inputs

- PSD or image file path
- New headline or text
- Output size and format
- Save or export-copy instruction

## Success Conditions

- The correct text layer or editable area was updated
- Layout remains readable after the change
- The final export uses the requested size and format

## Prompt Examples

- `Photoshopでバナー画像の見出しを差し替えて、PNGで書き出して。元PSDは上書きしないで。`
- `キャンペーン画像のテキストだけ4月版に更新して、Web用サイズで保存して。`

## Tips

- Mention the exact layer or visible text to replace if possible
- If font substitution breaks layout, stop before exporting and report the issue
- Prefer "Export a copy" for marketing assets when the source file should remain unchanged
