# Adobe Photoshop App Profile

Edit images precisely while protecting the source file and avoiding destructive raster changes without clear intent.

## Purpose

Photoshop is best for resizing, cropping, retouching, layer edits, and export for sales or marketing assets.

## Main Screens

- Canvas
- Layer panel
- Toolbar
- Properties and transform controls
- Export dialogs

## Typical Tasks

- Resize or crop an image
- Add simple text overlays
- Adjust canvas or image size
- Export PNG or JPEG versions

## Important Shortcuts

- `V`
- `C`
- `T`
- `Ctrl+T`
- `Ctrl+Alt+Shift+S`
- `Ctrl+Shift+S`

## Common Pitfalls

- Editing the wrong layer
- Moving a layer instead of the canvas selection
- Saving over the original PSD or source image
- Hidden transform handles or clipped text

## Operational Notes

- Confirm the active document and layer before modifying anything
- Prefer non-destructive actions, duplicate layers, and export copies when possible
- Re-observe after resize, crop, and text placement because control handles can hide content
- Match overwrite save and flattening behavior to the requested output format

## Prompt Tips

- Specify source image, target output size, output format, and whether to keep the original intact
- For marketing assets, mention target channel such as email, web banner, or social thumbnail
- If only export is needed, say “export a copy” rather than “save”
