# Paint App Profile

Launch Paint quickly, pick a simple drawing tool, and create small sketches with visible strokes.

## Purpose

Paint is best for quick line drawings, rough annotations, simple diagrams, and lightweight image edits.

## Main Screens

- Blank canvas
- Toolbar and tool strip
- Color palette
- Resize or save dialogs

## Typical Tasks

- Open Paint from Windows and start a blank canvas
- Choose Pencil or Brush
- Draw a simple icon, animal, or sketch
- Save or export a quick image

## Important Shortcuts

- `Ctrl+N`
- `Ctrl+S`
- `Ctrl+Z`
- `Ctrl+Plus`
- `Ctrl+Minus`

## Common Pitfalls

- Drawing before the canvas is active
- Switching tools unintentionally
- Tiny strokes that are hard to see

## Operational Notes

- Launch with `Win+R` and `mspaint` when a direct path is needed
- After Paint appears, focus on the canvas and pick Pencil or Brush if the current tool is unclear
- Plan drawings as a sequence of visible strokes with simple subgoals
- Use canvas-relative coordinates mentally, like a lightweight SVG plane, then convert each stroke into a multi-point drag path
- Approximate curves with several points instead of one straight drag
- Draw one visible stroke at a time and re-observe before the next stroke
- For simple animal sketches, start with head outline, ears, face, then body and tail

## Prompt Tips

- Say whether you want a rough sketch, line art, or simple annotation
- Mention the subject and how detailed it should be
- If you want cleaner curves, say `曲線は滑らかに、頭と体は丸みを付けて`
- If the task should stop before save, say so explicitly
