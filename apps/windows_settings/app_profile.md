# Windows Settings App Profile

## Purpose

System configuration surface with potentially broad machine impact.

## Main Screens

- Settings home
- Category pages
- Detail panes
- Confirmation dialogs

## Typical Tasks

- Check current settings
- Navigate categories
- Change configuration when it is the requested goal

## UI Structure

- Search box
- Category navigation
- Main detail pane

## Important Menus

- Search
- Category navigation
- Device or account-specific pages

## Important Shortcuts

- `Win+I`
- `Alt+Left`
- `Ctrl+L`

## Common Pitfalls

- Similar settings names
- System-wide impact
- Admin or account prompts

## Version Differences

- Page structure varies across Windows releases

## Operational Notes

- Change settings only on the exact page named or implied by the goal
- Observation-only navigation can proceed while gathering context

## Common Success Patterns

- Confirm exact page title before changing any toggle
- Confirm exact page title and target control before applying changes
