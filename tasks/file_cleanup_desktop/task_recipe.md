# Task Recipe: Desktop File Cleanup

Organize desktop or Downloads files into a clearer folder structure while minimizing accidental loss or overwrite.

## Best Used When

- Desktop files are cluttered
- Downloads should be grouped by type or project
- Temporary files should be separated from important files

## Inputs

- Source folder
- Grouping rule
- Destination folder names
- Delete or keep behavior

## Success Conditions

- Files are grouped under the intended structure
- Important files remain visible and recoverable
- Delete or overwrite actions follow the instruction exactly

## Prompt Examples

- `デスクトップ上のファイルを種類ごとに整理して、削除はせずフォルダ分けだけして。`
- `Downloadsフォルダを案件名ごとにまとめて、重複は残したまま整理して。`

## Tips

- Say whether duplicates should be skipped, renamed, or reviewed
- If no deletion is allowed, say so explicitly
- Give a grouping rule such as extension, month, client, or project
