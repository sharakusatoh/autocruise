# AutoCruise Prompt Library

AutoCruise CE でそのまま使いやすいプロンプト例と、実務で成功率を上げる書き方の Tips です。ソフト名、対象ファイル、終了地点、保存形式、フォルダ名を具体的に書くほど、画面探索と操作計画が安定します。

## まず押さえるコツ

- ソフト名、対象ファイル、対象フォルダ、対象画面を入れる。
- 「どこまで終わったら完了か」を明記する。
- 長い作業は「途中で状況を見ながら続けて」と書く。
- Web 作業はログイン済みか、どのブラウザを使うか、最終的に下書きか送信までかを書く。
- ファイル作業は、元ファイルを直接更新するのか、別名保存するのかを書く。
- 描画や編集作業は、形状、色、配置、キャンバス上の大まかな座標感を入れる。

## Office

### Excel

- `Excelで売上管理ブックを開いて、今月分の案件行を更新し、集計シートを見直して、保存前の状態まで進めて。`
- `Excelで請求一覧と入金一覧を照合して、不一致の行だけ色を付けて、差額メモ列を追加して。`
- `月次レポートのグラフを最新月まで更新し、画像として書き出して。`
- `在庫表の欠品行だけ抽出して、別シートに一覧を作って。`
- `営業パイプライン表の見込み金額を担当者別に集計し、合計行とフィルタを追加して。`

### Word

- `Wordで既存の見積書を開き、顧客名、金額、納期を更新し、PDF 出力前の状態まで整えて。`
- `提案書の導入効果セクションだけ文章を整え、元ファイルは直接更新せず別名保存して。`
- `会議メモを Word で議事録らしい体裁に整え、決定事項と次アクションを分けて。`
- `契約書の支払条件の項目だけ見直し、変更履歴が見える状態で止めて。`
- `報告書の表記ゆれと見出しだけ整え、本文の意味は変えずに仕上げて。`

### PowerPoint

- `PowerPointで提案資料の1枚目、材料ページ、導入事例ページを A社向けに更新して。`
- `週次報告スライドの数値とグラフだけ差し替えて、PDF 出力前の状態まで進めて。`
- `営業用ピッチ資料から不要な2枚を外し、役員向けの短い構成にして。`
- `プレゼン資料のロゴだけ新しいものに差し替え、レイアウト崩れがないか確認して。`
- `月次会議用の報告スライドを整え、タイトル表記のゆれも直して。`

### Outlook

- `Outlookで A社の田中さん宛てのフォローアップメールを下書きし、送信前の状態で止めて。`
- `未読メールのうち顧客名が入っているものを確認し、要返信だけフラグを付けて。`
- `受信トレイの通知メールを専用フォルダへ移し、重要メールだけ残して。`
- `明日の定例会を30分後ろ倒しにする更新メールを作成して。`
- `先週の打ち合わせのお礼メールを作り、資料を添付した状態で止めて。`

### Teams

- `Teamsで今日の打ち合わせチャットを見て、決定事項と宿題を Word にまとめて。`
- `会議後のフォローアップ要点を Teams から拾い、Outlook の下書きメールに入れて。`
- `Teams の会議メモから担当者ごとの ToDo を書き出し、見やすく整えて。`

### 技術

## ファイル整理

- `デスクトップのファイルを種類ごとに整理し、不要そうなものは削除せず一時フォルダへまとめて。`
- `ダウンロードフォルダの PDF だけを案件別に分け、元ファイル名は残したまま整理して。`
- `A社案件フォルダを作り、見積、契約、納品、請求の4フォルダを用意して。`
- `今月の領収書 PDF を日付付きファイル名に整え、経費フォルダへ移動して。`
- `古いスクリーンショットを年月フォルダに整理し、削除はせず移動だけして。`

## 技術者向け

### VS Code

- `VS Codeで old_service_name を検索し、置換候補を一覧で確認できるところまで進めて。`
- `ワークスペース全体で API_URL を検索し、src 配下だけ new URL に置換して。変更前に一度止めて。`
- `ログに出ているエラー文字列を VS Code で検索し、関連ファイルを開いて。`

### Terminal / PowerShell

- `Windows Terminalでこのプロジェクトの git status を確認してから、テストを実行して結果をメモ帳にまとめて。`
- `PowerShellを開いてビルドコマンドを実行し、失敗したらエラー行を探して。`
- `診断コマンドを順に実行し、生成されたログ zip の場所を確認して。`

## 書き方の型

- `ソフト名 + 対象 + 何をする + どこで完了する`
- `Excelで案件一覧.xlsxを開き、月次分を更新し、保存前の状態で止めて。`
- `Wordで提案書の導入効果セクションだけ整え、別名保存して。`
- `OutlookでA社向けフォローアップメールを下書きし、送信前で止めて。`
- `Edgeで競合比較を調べて、要点3つを Word へまとめて。`

## 失敗しにくくする一言

- `元ファイルは直接更新せず別名保存して。`
- `最終送信前で止めて。`
- `確認画面で止めて。`
- `見つからなければ別の起動方法を試して。`
- `1つずつ画面を確認しながら進めて。`
