# Adobe Acrobat App Profile

Review and annotate PDF documents with care around page order, comments, and export or save actions.

## Purpose

Acrobat is best for comment-based review, highlighting, page extraction, and PDF export workflows.

## Main Screens

- Document canvas
- Page thumbnail pane
- Comment tools
- Fill & sign or export tools

## Typical Tasks

- Open and review a PDF
- Add comments or highlights
- Extract pages
- Export to another format

## Important Shortcuts

- `Ctrl+O`
- `Ctrl+S`
- `Ctrl+Shift+S`
- `Ctrl+F`
- `Ctrl++`
- `Ctrl+-`

## Common Pitfalls

- Commenting on the wrong page
- Saving over the source file when a reviewed copy was intended
- Missing floating toolbars or hidden side panels
- Confusing Adobe Reader and Acrobat feature sets

## Operational Notes

- Confirm page number and visible page content before annotation
- Prefer comment and highlight tools for review tasks rather than editing text directly
- Export, page deletion, or flattening changes should match the requested output and be checked visually
- If comments are requested, include concise reviewer language rather than vague notes

## Prompt Tips

- Mention the document name, page range, and annotation style
- State whether to save in place, save a reviewed copy, or leave unsaved
- For technical reviews, specify what should be checked: wording, totals, missing pages, or formatting
