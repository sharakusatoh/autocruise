# Task Recipe: Excel Monthly Chart Update

Refresh monthly charts in Excel from existing tables while preserving the workbook layout and business meaning.

## Best Used When

- Monthly reporting slides depend on the latest Excel chart
- A dashboard must be updated for a sales or management review
- Trend lines or labels need a quick refresh

## Inputs

- Workbook name or path
- Sheet and chart names
- New month or date range
- Save or export instruction

## Success Conditions

- The correct chart or chart source table was updated
- Axis labels and legends still make sense
- The final workbook or exported image matches the requested month

## Prompt Examples

- `Excelで月次レポートの売上グラフを最新月まで更新して、画像として書き出して。`
- `ダッシュボードの案件推移グラフだけ直して、上書きせず別名保存して。`

## Tips

- Mention the chart title and source sheet if possible
- If several charts exist, name the one to update instead of saying "the graph"
- If the output will go into PowerPoint, say whether you want a workbook save, an image export, or both
