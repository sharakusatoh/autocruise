# Task Recipe: PowerPoint Customer Pitch Update

Update a customer-facing pitch deck in PowerPoint while preserving slide order, style, and message clarity.

## Best Used When

- A salesperson must tailor a deck for a specific account
- Pricing, logos, timelines, or case studies changed
- A short executive version needs to be prepared from an existing deck

## Inputs

- Deck file path
- Target customer or industry
- Slides to update
- Save or export instruction

## Success Conditions

- The intended slides were updated
- Layout and brand consistency remain acceptable
- The deck is saved or exported exactly as requested

## Prompt Examples

- `PowerPointで提案資料の1枚目、料金ページ、導入事例ページをA社向けに更新して。`
- `営業用ピッチ資料の不要な2枚を削って、PDF出力前で止めて。`

## Tips

- Specify slide numbers or titles whenever possible
- Mention whether text, chart, logo, or screenshot updates are needed
- If a PDF is required, say whether the `.pptx` should also be saved
