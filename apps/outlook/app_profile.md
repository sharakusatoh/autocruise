# Outlook App Profile

Handle email and calendar workflows with strong caution around recipients, attachments, and send actions.

## Purpose

Outlook is best for follow-up mail, meeting invites, mailbox triage, and simple scheduling support.

## Main Screens

- Inbox and folder list
- Reading pane
- Compose window
- Calendar view
- Meeting invite dialog

## Typical Tasks

- Draft a follow-up email
- Reply to a thread
- Create a meeting invite
- Move or categorize messages

## Important Shortcuts

- `Ctrl+R`
- `Ctrl+Shift+R`
- `Ctrl+N`
- `Ctrl+Enter`
- `Alt+S`

## Common Pitfalls

- Sending to the wrong recipient
- Replying to all when only one recipient is intended
- Attaching the wrong file
- Editing calendar details in the wrong timezone or date

## Operational Notes

- Draft or send according to the requested endpoint
- Verify recipients, subject, attachments, and send button state before any final send
- Prefer saving drafts or leaving compose windows open when send intent is unclear
- For calendar work, confirm date, timezone, and attendee list before completion

## Prompt Tips

- State whether the mail should be drafted only or actually sent
- Mention recipient names, subject line, and any required attachments
- For meeting invites, specify date, time, duration, attendees, and whether to send the invite or stop before sending
