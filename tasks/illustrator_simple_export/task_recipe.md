# Task Recipe: Illustrator Simple Export

Make a focused edit in Illustrator and export the requested asset while preserving the original working file.

## Best Used When

- A banner or flyer text needs a quick change
- A logo variant must be exported
- SVG or PNG assets are needed from an existing AI file

## Inputs

- AI file path
- Artboard or design target
- Required text or element change
- Export format and size

## Success Conditions

- The correct artboard is open
- Requested edit is visible
- Export file exists in the intended format

## Prompt Examples

- `Illustratorでバナーの見出しだけ差し替えて、PNGを書き出して。`
- `ロゴAIファイルからSVGを書き出して、元ファイルは上書きしないで。`

## Tips

- Mention artboard name or visible title
- Say whether typography, colors, and alignment must stay unchanged
- If multiple outputs are needed, list them explicitly
