# Chrome App Profile

## Purpose

General-purpose browser used through visible UI controls, UIA when available, and optional browser adapters when a connected page exists.

## Main Screens

- Tabbed browser window
- Address bar
- Download shelf or panels
- Permission prompts

## Typical Tasks

- Open pages
- Download files
- Copy information
- Interact with web UIs via visible controls

## UI Structure

- Tab strip
- Toolbar
- Content area
- Dialogs and permission sheets

## Important Menus

- Main menu
- Settings
- Downloads
- Print

## Important Shortcuts

- `Ctrl+L`
- `Ctrl+T`
- `Ctrl+W`
- `Ctrl+Shift+T`
- `Ctrl+J`

## Common Pitfalls

- Multiple similar tabs
- Permission prompts and downloads
- Content shifts and modal dialogs

## Version Differences

- Toolbar icons and prompt placement can change

## Operational Notes

- Complete uploads, form submissions, purchases, and account actions when they are clearly part of the goal
- Prefer visible controls first; use optional browser adapters only when a connected page is available

## Common Success Patterns

- Confirm the active tab title and visible controls
- Use shortcuts for address bar and tab management
