# Word App Profile

Create, revise, and format business documents while preserving layout and requested file handling.

## Purpose

Word is best for proposals, quotations, reports, minutes, and customer-facing documents with structured formatting.

## Main Screens

- Document canvas
- Ribbon
- Navigation pane
- Review / comments pane
- Save and export dialogs

## Typical Tasks

- Update an existing proposal or quote
- Insert tables, headings, bullets, and images
- Revise wording while keeping layout
- Export to PDF

## Important Shortcuts

- `Ctrl+S`
- `Ctrl+F`
- `Ctrl+H`
- `Ctrl+B`
- `Ctrl+Shift+C`
- `Ctrl+Shift+V`

## Common Pitfalls

- Typing into the wrong insertion point
- Breaking page layout with large paste operations
- Overwriting the wrong file
- Track Changes or comments affecting final output

## Operational Notes

- Confirm document title and insertion point before typing
- Prefer styles, headings, and table formatting over manual spacing
- If asked to revise a section, locate the exact heading first
- Save in place when the goal identifies the target document and output

## Prompt Tips

- Mention the document name, section name, and whether to save in place or save as a copy
- For sales work, specify client name, offer summary, and output format
- For long documents, ask for one section at a time instead of vague full-document edits
