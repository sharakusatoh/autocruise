# Task Recipe: File Project Folder Setup

Create a clean project folder structure in File Explorer for a new account, proposal, or engineering task.

## Best Used When

- A new sales opportunity needs standard folders
- An engineer starts a new investigation or delivery task
- Shared files must be organized before work begins

## Inputs

- Base location
- Project or customer name
- Folder template
- Whether existing folders may be reused

## Success Conditions

- The right base location was opened
- The required folders were created with the requested names
- Existing content was not overwritten unintentionally

## Prompt Examples

- `エクスプローラーでA社案件フォルダを作って、見積・提案・契約・納品の4フォルダを用意して。`
- `デスクトップ配下に検証用フォルダを作成し、logs、screenshots、notesに分けて。`

## Tips

- If the folder pattern is strict, list every subfolder name in the prompt
- Say whether date prefixes are needed
- If duplicates may already exist, tell it whether to merge, rename, or stop
