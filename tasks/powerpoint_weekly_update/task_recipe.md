# Task Recipe: PowerPoint Weekly Update

Refresh an existing deck with new weekly numbers, screenshots, or talking points while keeping slide order and style.

## Best Used When

- Weekly sales or project slides need revision
- A client deck needs text and image replacement
- A review PDF should be exported after edits

## Inputs

- Deck path or deck name
- Slide numbers or slide titles
- New bullet points, numbers, or screenshots
- Save or export instruction

## Success Conditions

- Target slides were updated
- Text fits on slide
- Images appear on the intended slide
- Deck is saved or exported as requested

## Prompt Examples

- `PowerPointの週次報告資料で3枚目と5枚目の数値を更新して、PDF出力前で止めて。`
- `営業提案資料の「導入事例」スライドだけ画像を差し替えて、既存デザインはそのまま維持して。`

## Tips

- Always mention slide number or slide title
- Say whether the result is draft, review-ready, or final
- If screenshots are involved, tell it where the source image is stored
