# Adobe Illustrator App Profile

Prepare vector graphics and exports with attention to artboards, layers, and export settings.

## Purpose

Illustrator is best for banners, simple layout edits, logo variations, and SVG or PNG asset export.

## Main Screens

- Artboard canvas
- Toolbar
- Layers panel
- Properties panel
- Export dialogs

## Typical Tasks

- Update text on a banner or flyer
- Resize or align objects
- Export SVG or PNG assets
- Duplicate artboards

## Important Shortcuts

- `V`
- `A`
- `T`
- `Ctrl+G`
- `Ctrl+Shift+O`
- `Ctrl+Shift+S`

## Common Pitfalls

- Selecting the wrong object or hidden layer
- Editing the wrong artboard
- Converting text to outlines accidentally
- Exporting the wrong artboard or scale

## Operational Notes

- Confirm artboard name or visible design before editing
- Prefer object selection and alignment tools over freehand dragging when precision matters
- Re-observe after text changes because overflow can be subtle
- Keep original AI files intact unless overwrite is explicitly requested

## Prompt Tips

- Mention artboard name, target text, export format, and output size
- For logo or banner work, state whether typography must stay unchanged
- If multiple artboards exist, say exactly which one to edit or export
